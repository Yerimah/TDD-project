# TDD project

## Built With

- Ruby

# Get Started
To get a local copy up and running follow these simple steps.

```bash
$ git clone https://github.com/yerimah/TDD-project
$ cd TDD-project

```

# Installation
Clone the repo and run bundle install in the command line to install the dependencies

```bash
- bundle install
- ruby main.rb
```
